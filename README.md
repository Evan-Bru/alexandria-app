# alexandria-app



alexandria_app/
│
├── app/                            # Pasta principal do app, onde fica a lógica e os componentes
│   ├── components/                 # Componentes reutilizáveis, como botões, cards, etc
│   ├── screens/                    # Telas do app, como Home, Catalogo, Detalhes do livro
│   ├── services/                   # Funções que lidam com dados e APIs (como leitura/escrita do Excel)
│   ├── utils/                      # Funções auxiliares (por exemplo, validações, formatação de dados)
│   ├── App.js                      # Arquivo principal que renderiza o app
│
├── assets/                         # Imagens, ícones, fontes, etc.
│   ├── images/                     # Imagens do app (ex: ícones de livros, capa de livro)
│   └── fonts/                      # Fontes customizadas
│
├── node_modules/                   # Dependências do projeto
│
├── package.json                    # Informações sobre o projeto, dependências e scripts
└── .gitignore                      # Arquivo que define quais arquivos não devem ser versionados
